InfoRec 軟體註冊與授權指南

感謝您採用 InfoRec 資訊紀錄管理工具。為了確保您的軟體權益與完整的保固服務，請參考以下流程完成註冊。

一、 準備工作：下載與安裝
(一) Windows 桌面工具 (InfoRec.exe)
請至官方網站 https://www.inforec.dev/download.html 下載並執行 InfoRec.exe。這是您進行資訊管理、數據儲存文件套印與各項服務的核心工具。
(二) 支付與授權工具 (區塊鏈錢包)
本系統採用 Solana 區塊鏈進行授權驗證，請依您的設備準備：
  • 1. Solana 手機用戶：請至 Solana dApp Store 下載「InfoRec App」。
  • 2. 一般電腦/手機用戶：請在瀏覽器安裝 Phantom 或 Solflare 錢包外掛，並透過 PWA 網頁版進行授權。
二、 註冊流程說明

本系統提供「線上授權」與「離線申請授權」兩種方式：

【方案甲：線上授權 (預設)】

支付完成後立即取得雲端授權，自動完成註冊。

步驟 1：於桌面工具提交申請
  • 開啟 InfoRec.exe，點選「支援 > 加入社群 > 申請註冊」。
  • 填寫用戶名稱、聯絡人、電子郵件與聯絡電話。
  • 選擇硬體指紋來源（電腦名稱或隨身碟磁區），點選「執行」。
  • 系統將同時產生：
    • 一般用戶：支付網址 (PWA)。
    • Solana 手機用戶：QR Code
步驟 2：使用錢包支付
  • 掃描 QR Code 或進入網址，選擇您的訂閱方案。
  • 於錢包視窗確認並授權傳送 SKR 代幣。
步驟 3：啟動軟體授權
  • 回到桌面工具，點選「支援 > 加入社群 > 執行註冊」。
  • 系統將自動驗證鏈上轉帳結果，確認無誤後即完成註冊。
【方案乙：離線申請授權】

適用於無法親自操作虛擬貨幣、或設備處於離線環境之用戶。

步驟 1：產生申請檔案
  • 於申請註冊視窗選擇「離線申請授權」。
  • 填寫資料後點選「執行」，系統將自動產生申請檔並引導您發送郵件給代辦者。
步驟 2：轉帳給代辦人
  • 若您無特定代辦人，請將金融匯款證明提供給本軟體管理人員，由管理人員進入「代辦模式」為您傳送 SKR 代幣並產生授權。
步驟 3：匯入授權檔完成註冊
  • 收到代辦人員回傳的「授權檔案」後，將其存入 InfoRec 程式資料夾。
  • 於註冊畫面勾選「檔案匯入註冊」，再點選「執行註冊」即可啟用。
三、 代辦模式

此模式用於協助他人完成 SKR 轉帳並取得授權檔案：

  1. 代辦人員開啟註冊視窗，選擇「代辦模式」。
  2. 點選讀取申請者提供的「授權申請檔案」。
  3. 由代辦人員之錢包完成 SKR 轉帳，系統將產生該設備專用的「授權檔案」。
  4. 代辦人員將檔案轉交給申請者，依「方案乙」步驟 3 匯入即可。
⚠️ 注意事項
  • 1. 【硬體綁定】註冊資料會與您的硬體設備關聯，採用電腦名稱註冊者，若需更換電腦，請務必將新電腦名稱修改為與原註冊名稱一致。
  • 2. 【退款條款】區塊鏈交易具有不可逆性，一旦成功後無法退款。請於付款前充分試用軟體，確認符合您的期待。
  • 3. 【保固延續】續約保固將由原到期日自動順延，不會縮短您既有的權益天數。
  • 4. 【技術支援】若有任何註冊或軟體使用上的問題，歡迎隨時與本軟體技術支援團隊（管理者）聯繫。

InfoRec Registration & Licensing Guide

Thank you for choosing InfoRec. To ensure your software rights and full warranty services, please complete the registration process below.

I. Prerequisites: Download & Installation
(1) Windows Desktop Tool (InfoRec.exe)
Please visit https://www.inforec.dev/download.html to download and run InfoRec.exe. This is your core tool for data management and services.
(2) Payment & Licensing Tools (Blockchain Wallet)
The system uses Solana blockchain for verification. Please prepare based on your device:
  • 1. Solana Phone Users: Download "InfoRec App" from the Solana dApp Store.
  • 2. Regular Users: Install Phantom or Solflare wallet extension and authorize via PWA version.
II. Registration Process

The system provides two methods: "Online Activation" and "Offline License Request":

【Option A: Online Activation (Default)】

Get cloud license immediately after payment to complete registration automatically.

Step 1: Submit Request via Desktop Tool
  • Open InfoRec.exe, go to "Support > Join Community > Register".
  • Fill in User Name, Contact, Email, and Phone Number.
  • Select Hardware Fingerprint source (PC Name or USB ID) and click "Execute".
  • The system generates:
    • Regular Users: Payment URL (PWA).
    • Solana Phone Users: QR Code
Step 2: Payment via Wallet
  • Scan QR Code or follow URL, then choose your subscription plan.
  • Authorize the SKR token transfer in your wallet.
Step 3: Activate Software License
  • Return to Desktop Tool, click "Execute Registration".
  • System verifies on-chain transaction and completes activation.
【Option B: Offline License Request】

For users unable to use crypto or devices in offline environments.

Step 1: Generate Request File
  • Select "Offline License Request", fill info, and click "Execute" to generate a request file.
Step 2: Transfer to Agent
  • If you do not have a specific agent, provide payment proof to our management staff. They will use "Proxy Mode" to process your license.
Step 3: Import License File
  • Place the received License File in the InfoRec folder.
  • Check "Import from File" and click "Execute Registration" to activate.
III. Proxy Mode

Used to assist others in completing SKR transfers and obtaining license files:

  1. Agent opens registration, selects "Proxy Mode".
  2. Load the ".req" file provided by the applicant.
  3. Complete payment via agent's wallet to generate a device-specific license.
  4. Deliver the file to the applicant for import (Step 3 of Option B).
⚠️ Important Notes
  • 1. [Hardware Lock] License is bound to your hardware. If changing PCs, PC name must match original.
  • 2. [Refund Policy] Blockchain transactions are irreversible. No refunds after payment.
  • 3. [Warranty Extension] Renewals automatically extend from original expiration date.
  • 4. [Technical Support] For any issues, contact the Technical Support Team (Admin).

  • 开启 InfoRec.exe，点击“支持 > 加入社区 > 申请注册”。

适用于无法亲自操作虚拟货币，或设备处于断网环境的用户。

步骤 1：生成申请文件

步骤 3：导入授权文件
  • 将接收到的授权文件放入目录，勾选“文件导入注册”并点击执行即可。

此模式用于协助他人完成 SKR 转账并获取授权文件：
