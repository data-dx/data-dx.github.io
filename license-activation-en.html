InfoRec Software Licensing & Activation Guide

To protect your rights regarding current software licensing, as well as existing version updates and download entitlements, please refer to the following guide to complete software function activation and verification.

Prerequisites: Software Download & Installation

  1. Download InfoRec (Information Logging Management System) Windows Desktop Tool

    Please visit the project download page https://www.inforec.dev/download.html to download and run InfoRec.exe. This is your core tool for information management, data storage, form template printing, and various function verifications.

  2. Licensing & Verification (Blockchain Technology Verification)
    1. Web3 Blockchain Online Licensing Verification

      This project utilizes the Solana blockchain for environment verification. Please choose one of the following paths based on your device to complete function activation verification:

      • Solana Phone Users: Please visit the Solana dApp Store to download the "InfoRec App", and install the Phantom wallet if necessary.
      • General Desktop Users: Please install the Phantom wallet extension in your browser and verify via the PWA web version.
      • Android Phone Users: Please download the APK file from this project website and install the Phantom wallet.
    2. Traditional Licensing Verification (Non-Blockchain)

      Execute "Request Offline License", and then obtain the license verification file generated through the "Assisted Offline Licensing" channel to complete the verification.

  3. Reference Links: Software Licensing & Activation, Apply for Software License

Process Description

This software provides two licensing methods: Online Licensing and Traditional Licensing. The brief descriptions are as follows:

  1. Online Licensing (Default Method)

    Users independently apply for a license and complete online Web3 blockchain verification and activation.

    • Submit Request via Desktop Tool
      Open InfoRec.exe, click "Support > Licensing & Activation > Apply for Verification > Verification Mode (Online Licensing)". Fill in your email address or entitlement identification alias, select the hardware binding source (PC name or USB drive partition), and click the "Execute Verification" button. The system will generate a QR Code containing the verification data URL (PWA).
    • Technical Verification via Blockchain
      Scan the QR Code or access the blockchain verification URL, then choose your software licensing plan. Submit the licensing activation to proceed. The technical verification mechanism in this process is designed to establish a minimum barrier to entry (anti-abuse mechanism) and serves as a symbolic reasonable consideration for intellectual property licensing.
    • Complete Environment Verification
      Return to the desktop tool, click "Support > Licensing & Activation > Execute Verification". The system will automatically compare the on-chain verification results, and complete the environment activation process once verified.
  2. Traditional Licensing Verification (Non-Blockchain) - Requires cooperation with an "Assisted Offline Licensing" user to complete verification

    Applicable to computers without an internet connection, or users with no experience in using blockchain (verified with the assistance of community members).

    • Generate Request File
      Open InfoRec.exe, click "Support > Licensing & Activation > Apply for Verification > Verification Mode (Request Offline License)". Fill in the required data and click "Execute Verification". The system will automatically generate a verification request file; please hand this file over to a community member for assistance with technical verification.
    • Transaction Verification via Community Peer Assistance
      If you have no experience using blockchain, a community member (third party) with a Web3 environment can execute "Assisted Offline Licensing" to complete the technical verification data and generate the verification file on your behalf.
    • Execute Local Verification
      • "License via File": For offline devices, after obtaining the license file, select "License via File" and then click "Execute Verification" to complete local environment activation.
      • "Direct Online Verification": For computers with an internet connection, directly click "Execute Verification" to read the data and complete verification.

Important Notes

  • Environment & Hardware Lock: This software is fixed "as-is" to the Windows operating system environment, and utilizes the PC name or USB drive partition volume serial number for hardware binding. For those using PC names for verification, when changing computers, please make sure to manually modify the new PC name to match the original verified name.
  • Download Consideration Disclosure: The blockchain technical verification mechanism aims to provide the software to users who genuinely need it, preventing automated programs from abusing software resources due to open access without barriers. Because blockchain transactions possess irreversible technical characteristics, once the environment verification is successful, tokens (such as SKR) used as a symbolic consideration for intellectual property licensing cannot be refunded.
  • License Period Extension: If environment activation and verification are continuously completed subsequently, the version update and derivative download entitlements will be automatically extended from the original expiration date.
  • Community Mutual Help & Troubleshooting: For any questions regarding software functionality or operations, you are welcome to submit them to the community peer assistance platform for experience sharing and self-troubleshooting.
