InfoRec Software Licensing and Activation Guide

To protect your software as-is licensing rights, as well as your existing version update and download privileges, please refer to the following process guide to complete software feature activation and verification.

Prerequisites: Software Download and Installation

  1. Download InfoRec (Information Records Management System) Windows Desktop Tool

    Please go to the project website download page at https://www.inforec.dev/download.html to download and run InfoRec.exe. This is your core tool for information management, data storage, document printing, and various feature verifications.

  2. Licensing and Verification: This software provides two methods for licensing verification: Web3 blockchain online licensing and traditional licensing verification.
    1. Web3 Blockchain Online Licensing Verification

      This project uses the Solana blockchain for verification. Please choose one of the following paths based on your device to complete the feature activation verification:

      • Solana Mobile Users: Please go to the Solana dApp Store to download the "InfoRec App" and install the Phantom wallet as needed.
      • Standard Computers: Please install the Phantom wallet extension in your browser and verify via the PWA web version.
      • Android Mobile Users: Please download the APK file from this project website and install the Phantom wallet.
    2. Traditional Licensing Verification (Non-Blockchain)

      Execute "Request Offline Licensing," and then obtain the license verification file generated through the "Assist Offline Licensing" channel to complete the verification.

  3. References: Software Licensing and Activation, Apply for Software License

Process Description

  1. Online Licensing (Default Method)

    Users independently apply for licensing and complete online Web3 blockchain verification and activation.

    • Submit Application via the Desktop Tool

      Open InfoRec.exe, and click "Support > Licensing & Activation > Request Verification > Verification Mode (Online Licensing)." Fill in your email address or rights identification alias, select the hardware binding source (Computer Name or USB Drive Partition), and click the "Execute Verification" button. The system will generate a QR Code containing the verification data URL (PWA).

    • Technical Verification via Blockchain

      Scan the QR Code or enter the blockchain verification URL, and select your software licensing plan. Submit the licensing activation to proceed. The technical verification mechanism in this process is designed to build a minimal barrier to entry (anti-abuse mechanism) and serves as a symbolic, reasonable consideration for intellectual property licensing.

    • Complete Licensing Verification

      Return to the desktop tool, and click "Support > Licensing & Activation > Execute Verification." The system will automatically compare the on-chain verification results and complete the activation process once confirmed.

  2. Traditional Licensing Verification (Non-Blockchain), requires cooperation with an "Assist Offline Licensing" user to complete verification

    Applicable to computers without an internet connection, or users without blockchain experience (verified through community collaboration).

    • Generate Application File

      Open InfoRec.exe, and click "Support > Licensing & Activation > Request Verification > Verification Mode (Request Offline Licensing)." Fill in the information and click "Execute Verification." The system will automatically generate a verification application file. Please hand this file over to a community member to assist with the technical verification.

    • Transaction Verification via Community Collaboration

      If you do not have blockchain experience, a community member (third party) with a Web3 environment can execute "Assist Offline Licensing" to complete the technical verification data and generate a verification file on your behalf.

    • Execute Local Verification
      • "License from File": For offline devices, after obtaining the licensing file, select "License from File" and then execute "Execute Verification" to complete local activation.
      • "Direct Online Verification": For computers with an internet connection, directly executing "Execute Verification" will read the data and complete the verification.

Precautions

  • Environment and Hardware Locking: This software is fixed as-is to the Windows operating system environment and uses the Computer Name or USB Drive partition number for hardware binding. For users verifying by Computer Name, when changing computers, please ensure you manually modify the new computer name to match the original verified name.
  • Download Consideration Description: The blockchain technical verification mechanism aims to provide the software to users who truly need it, preventing automated programs from abusing software resources due to a zero-threshold release. Because blockchain transactions possess irreversible technical characteristics, once licensing verification is successful, the tokens (such as SKR) used as symbolic consideration for intellectual property licensing cannot be refunded.
  • Extension of Update Validity: If licensing activation and verification continue to be completed successfully in the future, your version update and derivative download rights will be automatically extended from the original expiration date.
  • Community Mutual Aid and Troubleshooting: If you have any questions regarding software feature usage or operations, you are welcome to submit them to the community collaboration platform for experience sharing and independent troubleshooting.
