入門指南

以下為入門基本操作與流程：快速建立文件底圖、設定輸入欄位、列印並保存紀錄。 最新更新日期：2025/12/20。

一、安裝與啟動

  1. 下載 InfoRec 壓縮檔後解壓縮即可使用。
  2. 支援 Windows 全系列（含 Win11、LTSC）。
  3. 若 Windows 顯示不明發行者，可先允許執行或加入白名單。

二、產生文件底圖

為了能在畫面上定位欄位並套印，請先將紙本文件掃描或拍攝後：

  • 將 PDF 轉檔為 PNG（推薦 300 DPI）。
  • 透過「產生文件圖檔」工具裁切與調整大小。

三、建立文件資料

  1. 新增領域（如台灣、香港、國際）。
  2. 新增來源（如金融、行政、保險、教育）。
  3. 新增文件編號、版本及名稱。

四、設定表單輸入欄位

  • 放置輸入元件（文字、下拉、日期）。
  • 決定欄位位置、大小、群組。
  • 加入資料驗證（如長度、格式、轉換、運算）。

五、輸入 → 套印 → 保存紀錄

完成設計後，可直接輸入資料，系統會：

  • 填入文件底圖進行列印。
  • 保存輸入內容供日後追溯。
  • 支援查詢、月報、分類、追蹤。
