功能說明

請注意「前置字串」參數：若入「-」符號，系統會自動填滿未使用的欄位。例如設有「萬、千、百、拾、個」五個欄位，當輸入「100」時，會分別顯示為「- - 1 0 0」或「- - 壹 零 零」。

  1. 日期：(功能的參考日期，會呈現使用時的日期)
    • 每月一張：當一次要列印多張文件但月份不能相同時使用。例如一次印24張支票，作為銀行借款還款使用，此時在支票的日期加上這個選項，於主頁列印時，將選取「每月一張」，並顧「列印張數」輸入24，再按最印，此時就可以依輸入的日期(第1張)，每張日期會加上1個月，的連續印出24張不同月份的日期的支票。
    • 西元日期格式化（2026/7/13 → 2026-7-13）：將西元日期轉換為指定符號格式的日期。
    • 西元日期拆分（2026/7/13 → 2026 7 13）：將西元日期拆分為無分隔符號的年、月、日。
    • 臺灣日期格式化（2026/7/13 → 115-7-13）：將西元日期轉換為指定符號格式的臺灣民國日期。
    • 臺灣日期拆分（2026/7/13 → 115 7 13）：將西元日期拆分為無分隔符號的臺灣民國日期格式的年、月、日。
  2. 數字：
    • 數字千分位格式化（1000 → 1,000）：為數值資料加入千分位符號，並可指定前置字串如「$」或「-」、附加字串與指定的小數位數。
    • 數字由左至右拆分填碼（1000 → 1 0 0 0）：將數字依位數拆開，採取由左至右的順序填入指定的欄位，用於字串類的資料，如銀行帳號。
    • 數字由右至左拆分填碼（1000 → 1 0 0 0）：將數字依位數拆開，採取由右至左的順序填入指定欄位，用於數值類的資料，如金額。
    • 數字轉英文金額（1.23 → ONE AND 23/100 ONLY）：將數值轉換為標準英文大寫金融金額格式，可自由搭配前置或附加字串。
    • 數字轉簡體中文金額（1.23 → 1元2角3分）：將數字資料轉換為簡體中文金額表示法（含元、角、分之單位），並支援附加項目宣告。
    • 數字轉簡體大寫拆分（123 → 壹 贰 叁）：將數值轉換為簡體中文金融大寫數字，並將字元逐一拆分以利填入對應的欄位。
    • 數字轉香港中文金額（1.23 → 1元2角3分）：將數字轉換為香港習慣之中文金額表示法，精確處理元、角、分單位轉換。
    • 數字轉香港大寫拆分（123 → 壹 貳 叁）：將數值轉換為香港習慣之繁體金融大寫數字，並進行逐字拆分填碼。
    • 數字轉臺灣中文金額（1.23 → 1元2角3分）：將數值資料轉換為臺灣習慣之中文金額格式，可自行搭配前置與附加字串。
    • 數字轉臺灣大寫拆分（123 → 壹 貳 參）：將數字轉換為臺灣標準金融大寫數字（壹、貳、參），並自動進行逐字拆分。
    • 臺灣大寫萬元合併拆分（123456 → 壹拾貳 參 肆 伍 陸）：將數值轉換為臺灣金融大寫格式，並針對萬元以上或以下進行不同的合併與拆分，以對特殊文件進行套印。
    • 數字轉西班牙文金額（1.23 → UN CON 23/100 EUROS）：將數字轉換為標準西班牙文大寫金額格式。
    • 依間距值編號：讓文件列印產生流水號，例如「連續報表式標籤紙」件數為3，此時需要3-1、3-2、3-3印於那3張標籤紙，這個功能可以分別產生1、2、3的內容。
  3. 字串：
    • 直接套用：資料不做任何處理，直接使用。
    • 取得部份字串：依設定取得指定的資料。
    • 字串逐字拆分：將資料逐字分開，以供群組成員(欄位)使用。
    • 依分隔符號拆分：依據指定符號（如連字號「-」）將原始字串切分（例如將電話號碼「04-12345678」拆分為區域碼與電話號碼）。
    • 指定字數拆分：設定固定的字數上限，當原始字串長度超過該設定值時，系統即自動將其切割成多個段落或行數。
    • 多選一（依共享順序選擇）：文件上面有多個選項可以選擇，依使用者輸入的數字，於該位置印出指定符號，例如文件上有4個選項「現金、支票、信用卡、轉帳」，當輸入2時，就會勾選支票。
    • 移除尾字：將指定的字串去除。
    • 依條件移除尾字：例如套印時，信封上面地址項目如(鄕鎮市區)已印刷完成，當資料有該字串時如「台中市」，此功能就能將「市」去除，只套印出必要的字串(台中)。
    • 置中補字對齊：列印信封收件人置中列印時，如果人名太短，可先告知，收件人可列印的最大字數，並將插入字串設為空白，在不超過可列印範圍長度時，會在收信人每個字的中間插入空白。
    • 長度不足補齊： 指定目標長度，當原始資料長度不足時，可設定前綴與後綴字串，自動在原始資料的左邊補到指定長度。
