InfoRec 軟體授權與啟用說明

歡迎採用 InfoRec 軟體（以下簡稱 InfoRec）資訊紀錄管理工具。為了保障您的軟體現狀授權權益與既有版本更新之衍生下載權益，請參考以下流程引導完成軟體功能啟用與環境驗證登錄。

一、 準備工作：下載與安裝

  1. Windows 桌面工具 (InfoRec.exe)
    請至專案下載頁面 https://www.inforec.dev/download.html 下載並執行 InfoRec.exe。這是您進行資訊管理、數據儲存文件套印與各項功能驗證的核心工具。
  2. 驗證與授權工具 (區塊鏈錢包)
    本系統採用 Solana 區塊鏈進行授權與環境驗證，請依您的設備準備：
    • Solana 手機用戶：請至 Solana dApp Store 下載「InfoRec App」。
    • 一般電腦/手機用戶：請在瀏覽器安裝 Phantom 或 Solflare 錢包外掛，並透過 PWA 網頁版進行授權。

二、 流程說明

本系統提供「線上環境驗證」與「離線申請環境驗證」兩種方式：

  1. 【方案甲：線上環境驗證 (預設)】

    驗證完成後立即取得雲端授權，自動完成本機環境登錄。

    • 步驟 1：於桌面工具提交申請
      開啟 InfoRec.exe，點選「支援 > 加入社群 > 申請功能啟用」。填寫用戶名稱、聯絡人、電子郵件與聯絡電話。選擇硬體指紋來源（電腦名稱或隨身碟磁區），點選「執行」。系統將同時產生一般用戶的支付網址 (PWA) 或 Solana 手機用戶的 QR Code。
    • 步驟 2：使用錢包進行環境與技術驗證
      掃描 QR Code 或進入網址，選擇您的智慧財產權授權方案。於錢包視窗確認並採用 SOL 區塊鏈的 SKR 傳送驗證，此技術驗證機制旨在建構最低經濟屏障（防濫用機制），並作為智慧財產權授權之象徵性合理對價。
    • 步驟 3：啟動軟體授權
      回到桌面工具，點選「支援 > 加入社群 > 執行環境驗證」。系統將自動比對鏈上轉帳與環境驗證結果，確認無誤後即完成功能啟用。
  2. 【方案乙：離線申請環境驗證】

    適用於無法親自操作虛擬通貨、或設備處於離線環境之用戶。

    • 步驟 1：產生申請檔案
      於功能驗證視窗選擇「離線申請授權」。填寫資料後點選「執行」，系統將自動產生申請檔並引導您發送郵件給社群協力人員。
    • 步驟 2：透過社群協力轉帳驗證
      若您無特定社群協助對象，請將相關證明提供給社群協力人員，由社群協力節點進入「代辦模式」採用 SOL 區塊鏈的 SKR 傳送驗證，為您代轉並產生技術授權檔案。
    • 步驟 3：匯入授權檔完成本機環境驗證
      收到社群協力人員或驗證節點回傳的「授權檔案」後，將其存入 InfoRec 程式資料夾。於啟用畫面勾選「檔案匯入啟用」，再點選「執行環境驗證」即可完成啟用。

三、 代辦模式

此模式用於社群協力協助他人完成 SKR 轉帳驗證並取得特定環境之授權檔案：

  1. 社群協力人員開啟驗證視窗，選擇「代辦模式」。
  2. 點選讀取申請者提供的「授權申請檔案」。
  3. 由社群協力人員之錢包完成 SKR 轉帳驗證，系統將產生該設備與環境專用的「授權檔案」。
  4. 社群協力人員將檔案轉交給申請者，依「方案乙」步驟 3 匯入即可。

四、 注意事項（重大事項揭露與免責宣告）

  • 1. 【環境與硬體鎖定】本軟體永久授權與特定作業系統環境及硬體設備相依。採用電腦名稱進行環境驗證者，用戶須具備獨立部署與操作能力，若更換電腦環境，請務必自行將新電腦名稱修改為與原驗證名稱一致。本軟體技術性質純粹，不提供人工環境重設、轉移或其他個別化技術諮詢勞務。
  • 2. 【退款條款】區塊鏈交易具有不可逆性，驗證成功後無法退款。此技術驗證機制旨在建構最低經濟屏障，以防止惡意自動化程式大量刷取、濫用軟體資源，並作為智慧財產權授權之象徵性合理對價。請於驗證前充分試用軟體，確認符合您的期待。
  • 3. 【更新效期延續】續約之既有版本更新衍生下載權益將由原到期日自動順延，不會縮短您既有的權益天數。
  • 4. 【社群互助與取得支援】若有任何軟體功能使用或環境相依性問題，歡迎提交至社群協力平台進行經驗分享與問題解決。本軟體定位為智慧財產權授權行使，不提供任何常態性人工客服、定期維護或售後技術支援服務。如需進一步瞭解技術規範，請尋求協助。指向特定排版調整時，請參閱 列印調整說明。
